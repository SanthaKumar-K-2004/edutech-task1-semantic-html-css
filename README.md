## Interview Questions - Quick Notes
1. Difference between semantic and non-semantic elements:
   - Semantic elements describe meaning and purpose (for example: article, section, nav).
   - Non-semantic elements are generic containers (for example: div, span).
2. CSS Box Model:
   - Every element is made of content, padding, border, and margin.
